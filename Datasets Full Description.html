Introduction

The DIVINE-datasets provides a collection of curated, anonymized datasets derived from the DIVINE project. These datasets include various clinical domains—such as patient demographics, comorbidities, symptoms, laboratory findings, treatments, and outcomes—allowing users to explore and practice different data manipulation and analysis workflows in a realistic setting.

Available Datasets

This repository includes the following datasets:

Dataset Description
analytics Laboratory parameters at hospital admission.
comorbidities Baseline comorbid conditions and functional indices.
complications In-hospital complications during COVID-19.
concomitant_medication Chronic treatments before hospital admission.
demographic Baseline demographic and lifestyle data.
end_followup Outcomes and follow-up information.
icu ICU admissions and critical care interventions.
inhosp_antibiotics In-hospital antibiotic treatments.
inhosp_antivirals In-hospital antiviral treatments.
inhosp_other_treatments Other in-hospital supportive therapies.
scores Disease severity scores at admission.
symptoms Clinical symptoms at hospital admission.
vital_signs Vital parameters and oxygenation data.
vaccine COVID-19 vaccination status.

analytics

Dimensions: 5813 rows × 9 columns

Variable Description
record_id Identifier of each record (anonymized).

d_dimer D-dimer (µg/L).
c_reactive_protein C-reactive protein (mg/L).
